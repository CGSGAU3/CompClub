Инструкция по сборке программы:
Сборка осуществляется средством автоматизации сборки cmake и делается следующими двумя командами:

cmake -B build - генерирует файлы проекта
cmake --build build - непосредственно сборка на основе файлов проекта

Пояснения к тестовым примерам:
test.txt - валидный пример из задания
test_err1.txt - количество столов не число
test_err2.txt - ошибка в указании времени
test_err3.txt - в теле сообщения допущена ошибка (в данном случае - слишком много аргументов)
test_err4.txt - неправильный идентификатор сообщения
